NGROK内网穿透利器
===

1. 官网下载：

Secure tunnels to localhost

<https://ngrok.com/>

2. 注册获取Auth token

    ngrok authtoken  token
    Authtoken saved to configuration file: C:\Users\userName/.ngrok2/ngrok.yml

3.本地服务CMD：

ngrok http 8080

ngrok -subdomain=weixin 8080

4.查看所有请求

http://localhost:4040/inspect/http
